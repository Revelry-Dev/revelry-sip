Automatically Stick your Front Page and Blog Posts page to the top of the pages list.

== Description ==

This plugin automatically sticks your front page and plog posts page to the top of the pages list if there is no manual orderby set.  If there are any children of the Front page, those will be listed after the Front page like normal, and then the page for posts.

== Installation ==

1. Download the zip file and upload it via the WP Admin Plugins page or upload it to the plugins directory 
2. Activate the plugin through the Plugins menu in WordPress 
3. That's it!  

== Frequently Asked Questions ==

= Is there a settings page? =

There is not as the plugin automatically re-orders upon activation.
